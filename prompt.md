- Мне нужно сделать функцию на языке Java для преобразования фрагментов HTML в текст (plain text). Текстовый формат мне нужен для того, чтобы включить эту выходную строку в состав элемента JSON объекта. Этот JSON объект будет использован для сериализации в REST API.

- Входной фрагмент HTML - это не полный HTML код страницы с элементами \<HTML\>, \<BODY\> и так далее, а относительно небольшой фрагмент, в котором могут содержаться несколько абзацев текста \<p\>, и присутствовать базовая разметка для списков и стилизации текста - bold, italic и т.п. 

- Конечно, в текстовом формате невозможно сохранить стилизацию - поэтому стили текста будут утеряны. Однако, нужно сохранить форматирование абзацев и переводов строк <br>. 

- Входной HTML, в с соотвествии со стандартом HTML, может содержать ряд escape-последовательностей типа \n \t \r, множественные пробелы и переводы строк и так далее - их браузер игнорирует, и визуально они в браузере не видны. При конвертировании нужно это учесть.

- В дальнейшем нужно будет добавить функцию для конвертирования HTML фрагментов в Markdown
